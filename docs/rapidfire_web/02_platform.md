# Platform Foundation

RapidFire Web is built on the **ASP.NET Core platform**, leveraging:
- high performance
- cross-platform support
- proven reliability

Key Benefits:
- low memory footprint
- modular architecture
- cloud & container ready
- superior runtime performance

Developers can deploy easily on:
- Linux
- Windows
- Docker
- Kubernetes
