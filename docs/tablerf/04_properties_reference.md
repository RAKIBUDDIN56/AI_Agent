| Property                         | Type     | Description                                     |
| -------------------------------- | -------- | ----------------------------------------------- |
| **Model**                        | Property | The model class used as table source.           |
| **DataList**                     | Property | Bind table using a custom list instead of DB.   |
| **EditModel**                    | Property | Separate model for edit mode.                   |
| **Edit**                         | Property | Show/Hide edit buttons.                         |
| **Delete**                       | Property | Show/Hide delete buttons.                       |
| **ActionStyle**                  | Property | Inline (default), Popup, or Float action icons. |
| **ActionWidth**                  | Property | Width of action column.                         |
| **ActionText**                   | Property | Header text of action column.                   |
| **AfterDelete**                  | Method   | Callback executed after deleting a record.      |
| **Context**                      | Property | Custom alias for the model inside templates.    |
| **Filter**                       | Property | Lambda filter for restricting data.             |
| **SQL**                          | Property | Custom SQL query.                               |
| **PageSize**                     | Property | Number of rows per page.                        |
| **PageNo**                       | Property | Default page number for initial load.           |
| **Add / AddUrl / AddButtonText** | Property | Controls add-entry button behavior.             |
| **Height**                       | Property | Custom table height.                            |
| **LazyLoad**                     | Property | Load data only on demand.                       |
| **EditUrl**                      | Property | URL used for row editing navigation.            |
